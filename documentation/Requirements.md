Health/Progress indicators

## User Stories

As a competitive player I want to be challenged so I can have fun
When I open the game I want to be given multiple difficulty options

As a casual player I want to be able to have a chill experience so I can have fun
When I open the game I want to be given multiple difficulty options

As a visually impaired player, I want to be able to increase the UI size, so I can read it better
Given I start the game, when I open the settings, I want the option to change the font size

As a hard of hearing player, I want an option to change the audio volume, so I can hear the game
When I open the settings, I want an option to change the volume

As a player, I want to clearly read the UI, so that I can understand what i going on
When I play the game, The Interface should be readbale and easily understandable

As a player, I want the Art and Music to look/sound good, So that I can enjoy the game
When I play the game, I don't want any of the sprites to overlp weirdly and layered correctly, and I want the music to allign with the gameplay and be balanced correctly

As a player, I want good progress Indicators So that I can measure my progress through the game
When I play the game, I should be able to see the health of the zombies and the progress through the day.
